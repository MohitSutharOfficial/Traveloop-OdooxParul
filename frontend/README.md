# Traveloop Frontend

React-based frontend for Traveloop, an intelligent travel planning platform.

## Stack

- Vite
- React 18
- TypeScript
- Tailwind CSS
- Supabase Auth
- Lucide React

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Required Environment

```env
VITE_SUPABASE_URL=your_supabase_project_url
VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
```
